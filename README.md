# Travel Recommendation

## Overview
This is an Application to help travellers explore exciting destinations around the world.

## Features
- Standard methods use for DOM Manipulation
- Adding search results cards dynamically
- Allowing multiple variations of words making search more flexible

## Tech Stack
- HTML
- CSS
- JavaScript

## Project Flow
1. User opens the website link.
2. User clicks on search input.
3. User adds some text related to what he is exploring and clicks search.
4. Cards are displayed based on user input.

## Installation & Setup
Clone the repository
git clone https://github.com/JayaNeem/travelRecommendation.git

Navigate into the folder
cd travelRecommendation

Open index.html
