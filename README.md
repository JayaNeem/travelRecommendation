# Travel Recommendation

## Overview
This is an Application to help travellers explore exciting destinations around the world.

## Features
- Standard methods use for DOM Manipulation
- Adding search result cards dynamically
- Allowing multiple variations of words making search more flexible

## Tech Stack
- HTML
- CSS
- JavaScript
- DOM

## Project Flow
1. User opens the website link.
2. User adds some text related to what he is exploring in the search input and clicks search.
3. Listener attached to the button executes the function which fetches the recommendation from json, creates card for each recommendation and append them to a result div.
4. User Clicks on clear button.
5. Listener attached executes a function which clears input element and result div.
